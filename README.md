# タスク管理アプリ（Python GUI）

## 概要

日々のタスクや作業進捗を、チェックボックスや進捗バーを使って視覚的に管理できるPython製デスクトップアプリです。  
シンプルな設計で、個人のToDo管理から軽微なプロジェクト進捗まで対応可能です。

## 特徴

- ✅ タスクの追加・削除・編集に対応
- 📊 進捗バーで現在の完了率を表示
- 📂 「未完了」「完了」タブでタスク状態を分類（※開発中 or 実装済みに応じて調整）
- 💾 ローカルファイルにタスク情報を保存（JSON形式）
- 🖱 GUI操作で直感的に扱える

## 使用技術

- Python 3.10
- Tkinter（GUI構築）
- 標準ライブラリ（json, os など）
